# Python Data Types Cheat Sheet

## Numeric Types
```python
# Integer
x = 10
# Floating-point
y = 10.5
# Complex number
z = 3 + 4j
```

## Sequence Types
```python
# List (Ordered, Mutable)
lst = [1, 2, 3]
# Tuple (Ordered, Immutable)
tpl = (1, 2, 3)
# Range (Sequence of Numbers)
rng = range(5)
```

## Text Type
```python
# String
s = "Hello"
```

## Set Types
```python
# Set (Unordered, Unique Elements)
st = {1, 2, 3}
# Frozen Set (Immutable Set)
fst = frozenset({1, 2, 3})
```

## Mapping Type
```python
# Dictionary (Key-Value Pairs)
dct = {"key": "value"}
```

## Boolean Type
```python
# Boolean
b = True
```

## Binary Types
```python
# Bytes (Immutable Byte Sequence)
bts = b"hello"
# Bytearray (Mutable Byte Sequence)
ba = bytearray(5)
# Memory View (Memory View Object)
mv = memoryview(b"hello")
